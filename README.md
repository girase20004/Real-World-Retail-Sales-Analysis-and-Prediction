# Real-World-Retail-Sales-Analysis-and-Prediction

## Overview
This project performs retail sales analysis and sales prediction using the Superstore dataset.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Features
- Data Analysis
- Data Visualization
- Correlation Analysis
- Sales Prediction
- Machine Learning

## Visualizations
- Sales by Category
- Profit by Category
- Sales Distribution
- Sales vs Profit
- Correlation Heatmap

## Machine Learning
- Linear Regression
- Model Evaluation using R2 Score and MSE

## Learning Outcomes
- Retail Data Analysis
- Business Insights
- Predictive Modeling
- Data Visualization
